# CustomSearchSpotlight

A modern, customizable search application for Windows with AI integration (Google Gemini), voice support, and a beautiful glass/acrylic UI.

## Features

- **Glass/Acrylic UI**: Beautiful transparent window with blur effects
- **AI Integration**: Google Gemini AI for intelligent responses
- **Local Search**: Search files, folders, and applications using Windows Search
- **Voice Support**: Speech-to-text and text-to-speech (coming soon)
- **Hotkey**: Activate with Alt+Space (customizable)
- **Customizable**: Themes, opacity, hotkeys, and more
